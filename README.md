# 📰 Fake News Detection System

A simple Machine Learning-based Fake News Detection system built using **Python**, **NLTK**, and **Scikit-learn**.  
This application classifies news as **REAL** or **FAKE** using Natural Language Processing (NLP) techniques.

---

## 📌 Features

- Preprocesses text using NLP techniques  
- Removes stopwords and special characters  
- Converts text into numerical form using TF-IDF  
- Trains a Logistic Regression model  
- Predicts whether news is REAL or FAKE  
- Interactive command-line interface  

---

## 🛠️ Technologies Used

- Python  
- NLTK (Natural Language Toolkit)  
- Pandas  
- Scikit-learn  
- Regular Expressions (`re`)
